Introduzione

Cos'è Gemini CLI

Un agente AI open source direttamente nel terminale, alimentato dai modelli Gemini di Google.

Interfaccia terminale

Agente AI interattivo che gira nel tuo terminale, senza bisogno di browser o UI separata.

Open source

Codice pubblico su GitHub, licenza Apache 2.0. Modificabile e adattabile alle esigenze del team.

Powered by Gemini 2.5 Pro

Accesso gratuito a Gemini 2.5 Pro tramite Google AI Studio, il modello più capace di Google.

Funzionalità

Funzionalità principali

Tutto ciò che Gemini CLI può fare direttamente dalla riga di comando.

Comprensione del codice

Analizza interi repository, spiega architetture complesse e risponde a domande sul codebase.

Generazione e refactoring

Scrive nuovo codice, refactora funzioni esistenti e suggerisce ottimizzazioni contestuali.

Ricerca nel filesystem

Trova file, cerca pattern nel codice e naviga la struttura del progetto tramite linguaggio naturale.

Shell commands

Esegue comandi shell, interpreta output e automatizza task ripetitivi con supervisione AI.

Web search integrato

Cerca informazioni aggiornate sul web e integra i risultati direttamente nelle risposte.

Estensioni MCP

Supporta il Model Context Protocol per connettere tool esterni: database, API, sistemi interni.

Use cases

Nel workflow quotidiano

Come il team SYS-FASHION può integrare Gemini CLI nelle attività di sviluppo di tutti i giorni.

Mattina — Code review assistita

Incolla una PR e chiedi a Gemini di evidenziare rischi, pattern anomali e suggerire miglioramenti prima della review umana.

Durante lo sviluppo — Pair programming

Usa Gemini come secondo cervello: spiega un bug, esplora soluzioni alternative, genera boilerplate per nuove feature.

Debug — Analisi di log e errori

Fornisci stack trace e log di errore: Gemini li analizza nel contesto del codebase e propone una causa radice.

Fine sprint — Documentazione automatica

Genera changelog, aggiorna README, scrivi ADR partendo dal codice modificato e dai commit del branch.

Getting started

Come iniziare

Tre passaggi per avere Gemini CLI operativo sul proprio ambiente di sviluppo.

Step 1
Installazione
  • Node.js 18+ installato
  • npm install -g @google/gemini-cli
  • Verifica: gemini --version
Step 2
Autenticazione
  • Account Google gratuito
  • Esegui gemini auth
  • Autorizza nel browser
  • API key da AI Studio (opzionale)
Step 3
Primo utilizzo
  • Naviga nel tuo progetto
  • Lancia gemini
  • Prova: "Spiega la struttura di questo repo"
  • Esplora i comandi con /help
